# 빠른 시작 가이드

## 1. 앱 실행하기

```bash
python app.py
```

브라우저에서 열기: http://localhost:5000

## 2. 문제 풀기

### 단계 1: 문제 설정
1. **카테고리 선택**: 원하는 카테고리에 체크
   - "전체 선택" / "전체 해제" 버튼으로 빠르게 선택 가능
2. **문제 수**: 원하는 개수 입력 (기본값 10)
3. **키워드 검색** (선택사항):
   - "그룹 추가" 버튼으로 검색 그룹 추가
   - 같은 그룹 내 여러 키워드 = AND 검색
   - 다른 그룹들 = OR 검색
   - 예시:
     - 그룹1: [S3, 버킷] → S3과 버킷을 모두 포함하는 문제
     - 그룹2: [VPC] → 또는 VPC를 포함하는 문제

### 단계 2: 문제 풀이
1. 문제 읽기
2. 답 선택:
   - **단일 선택**: 라디오 버튼 중 하나 선택
   - **다중 선택**: 체크박스 여러 개 선택 가능
3. "정답 확인" 버튼 클릭
4. 정오답과 해설 확인
5. "다음" 버튼으로 다음 문제로 이동

## 3. 오답노트 보기

### 방법 1: 풀이 완료 후
- 마지막 문제에서 "다음"을 누르면 자동으로 오답노트로 이동

### 방법 2: 상단 네비게이션
1. "오답노트" 클릭
2. 회차 목록에서 보고 싶은 회차의 "보기" 버튼 클릭
3. 오답 문제와 해설 확인

## 팁

- 같은 카테고리에서 여러 회차 풀기 가능
- 키워드 검색은 문제 텍스트와 선택지 텍스트 모두에서 검색됨
- 다중 선택 문제는 정답을 모두 맞춰야 정답 처리됨 (순서 상관 없음)
- 세션 데이터는 서버에 저장되므로 페이지 새로고침해도 진행 상태 유지됨

## 문제 해결

### 문제가 로드되지 않음
→ 선택한 카테고리와 키워드 조건을 만족하는 문제가 없을 수 있습니다. 다시 설정해보세요.

### 정답 확인 버튼이 작동 안 함
→ 최소 하나의 답을 선택해야 합니다.

### 오답노트가 비어있음
→ 모든 문제를 맞혔습니다! 축하합니다! 🎉
